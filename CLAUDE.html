PureVex - 项目架构文档 ​

初始化时间： 2025-09-07 00:11:56
包管理器： Bun
技术栈： VitePress + TypeScript + Vue + AI分析

项目概述 ​

这是一个基于 VitePress 构建的智能极简博客主题，专注于开发技术、系统运维、区块链、安全防护和交易策略等领域的技术文章分享，项目使用 Bun 作为包管理器，文章内容简炼，使用中文逗号和句号，段落结尾不添加句号

系统架构 ​

核心模块 ​

🎨 VitePress 主题模块 (.vitepress/) ​

路径： .vitepress/
功能： 自定义博客主题和配置

核心文件 ​

  • config.ts - 站点配置，包含导航、搜索、评论等设置
  • theme/index.ts - 主题入口，注册全局组件
  • theme/serverUtils.ts - 服务器端工具函数
  • theme/custom.css - 自定义样式

Vue 组件 (.vitepress/theme/components/) ​

  • NewLayout.vue - 主布局组件
  • Archives.vue - 归档页面
  • Category.vue - 分类页面
  • Tags.vue - 标签页面
  • Comment.vue - 评论组件
  • Page.vue - 页面组件
  • FriendLinks.vue - 友情链接

工具模块 (.vitepress/utils/) ​

  • ai-analyzer.mjs - AI 分析引擎
  • linkHealthChecker.ts - 链接健康检查
  • simpleLinkChecker.ts - 简单链接检查

🤖 AI 分析系统 ​

路径： scripts/, .vitepress/utils/
功能： 智能文章分类和标签生成

核心功能 ​

  • 文章分类： 自动识别 Development、System、Blockchain、Security、Trading、Others
  • 标签生成： 基于内容分析生成技术栈、功能、工具标签
  • 内容优化： 提供文章质量分析和改进建议

配置文件 ​

  • .vitepress/ai-config.yaml - AI 分析配置
  • scripts/blog-ai.js - AI 命令行工具

📝 内容管理 ​

路径： posts/, pages/
功能： 文章和页面内容管理

文章结构 (posts/) ​

每篇文章包含标准 frontmatter：

yaml
---
date: 2024-12-18T00:00:00.000Z
title: 文章标题
category: 分类
tags: [标签1, 标签2]
description: 文章描述
ai_edited: true  # AI 编辑状态
ai_analyzed_at: '2025-09-06T15:33:44.261Z'
---

页面结构 (pages/) ​

  • archives.md - 文章归档
  • category.md - 分类页面
  • tags.md - 标签页面
  • topics.md - 主题页面
  • about.md - 关于页面

🎯 静态资源 ​

路径： public/
功能： 静态文件管理

目录结构 ​

  • img/cheat/ - 作弊截图
  • img/trading/ - 交易相关图片
  • favicon.ico - 站点图标

开发工作流 ​

📦 包管理 ​

bash
# 使用 Bun 作为包管理器
bun install          # 安装依赖
bun run dev         # 开发服务器
bun run build       # 构建生产版本
bun run preview     # 预览构建结果

🤖 AI 工具链 ​

bash
# AI 分析工具
bun run ai                 # AI 帮助
bun run ai:analyze        # 分析文章
bun run ai:reset          # 重置 AI 状态
bun run ai:status         # 查看 AI 状态

📊 开发环境 ​

bash
# 开发模式
bun run dev              # 标准开发
bun run dev:hidden       # 显示隐藏文章
bun run dev:ai           # 启用 AI 分析

技术规范 ​

🎨 代码风格 ​

  • TypeScript： 严格类型检查
  • Vue 3： Composition API
  • ESLint + Prettier： 代码格式化
  • VitePress： 静态站点生成

📝 文章规范 ​

  • 语言： 中文为主，技术术语保留英文
  • 标点： 使用中文逗号和句号
  • 段落： 段落结尾不添加句号
  • 结构： 清晰的层次结构和代码示例

🏷️ 标签体系 ​

  • 技术栈： python, javascript, typescript, lua, shell
  • 功能： tutorial, config, debugging, optimization
  • 工具： nvim, git, docker, kubernetes, vitepress
