页面管理模块 ​

模块路径： pages/
上级导航： PureVex → 页面管理模块

模块概述 ​

页面管理模块负责 PureVex 的静态页面管理，包括归档、分类、标签、主题和关于等页面的内容组织和展示，这些页面为用户提供了内容导航和博客信息的重要入口

页面结构 ​

mermaid
graph TB
    A[页面管理] --> B[导航页面]
    A --> C[内容聚合]
    A --> D[信息页面]
    
    B --> B1[topics.md - 主题页面]
    B --> B2[about.md - 关于页面]
    
    C --> C1[archives.md - 文章归档]
    C --> C2[category.md - 分类浏览]
    C --> C3[tags.md - 标签页面]
    
    D --> D1[站点信息]
    D --> D2[作者介绍]
    D --> D3[联系方式]

核心页面 ​

📚 归档页面 (archives.md) ​

功能： 按时间顺序展示所有文章

主要特性：

  • 按年月分组显示文章
  • 显示文章发布日期
  • 提供快速导航链接
  • 支持分页浏览

实现方式：

vue
<!-- 通过 Archives.vue 组件实现 -->
<Archives :posts="posts" />

🏷️ 分类页面 (category.md) ​

功能： 按分类展示文章

主要特性：

  • 显示所有文章分类
  • 统计每个分类的文章数量
  • 提供分类筛选功能
  • 支持分类间的快速切换

分类系统：

  • Development (开发技术)
  • System (系统运维)
  • Blockchain (区块链)
  • Security (安全防护)
  • Trading (交易策略)
  • Others (其他)

🏷️ 标签页面 (tags.md) ​

功能： 标签云和标签筛选

主要特性：

  • 标签云可视化展示
  • 标签使用频率统计
  • 点击标签筛选相关文章
  • 支持多标签组合筛选

🎯 主题页面 (topics.md) ​

功能： 技术主题导航

主要特性：

  • 按技术主题组织内容
  • 提供主题间导航
  • 展示主题相关文章
  • 支持主题深度浏览

👤 关于页面 (about.md) ​

功能： 博客和作者信息

主要特性：

  • 博客介绍和定位
  • 作者背景介绍
  • 技术栈展示
  • 联系方式和社交链接

🎨 页面组件 ​

对应组件关系 ​

页面文件组件功能
archives.mdArchives.vue文章归档展示
category.mdCategory.vue分类浏览
tags.mdTags.vue标签云和筛选
about.mdPage.vue通用页面展示

组件通信流程 ​

mermaid
sequenceDiagram
    participant P as 页面
    participant C as 组件
    participant D as 数据层
    participant V as 视图
    
    P->>C: 加载页面组件
    C->>D: 请求文章数据
    D->>D: 处理和过滤数据
    D->>C: 返回结构化数据
    C->>V: 渲染页面内容
    V->>U: 展示给用户
